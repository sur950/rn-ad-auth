# rn-ad-auth
A React Native Module which provides complete Authentication flow for Azure AD
## Installation

```sh
npm install rn-ad-auth
```

## Usage

```js
import { multiply } from "rn-ad-auth";

// ...

const result = await multiply(3, 7);
```
